# Feature: AI 자동 요약

## 기능 설명
URL 저장 즉시 자막 기반으로 OpenAI gpt-4o-mini가
3줄 요약 + 핵심 인사이트를 생성해 상세 패널에 표시한다.

> ⚠️ Claude API 사용 금지. 반드시 OpenAI gpt-4o-mini 사용.

## 출력 형식
- **3줄 요약**: 핵심 내용 3문장, 각각 독립적으로 읽힘
- **핵심 인사이트**: 불릿 3~5개, 실용적/행동 가능한 내용

## 표시 방식
- 생성 중: shimmer 스켈레톤 (스트리밍 방식 사용하지 않음)
- 생성 완료: 스켈레톤 → 콘텐츠 fade-in 전환
- 실패: "요약 생성 실패 - 재시도" 버튼

## 실행 흐름
1. URL 저장 트리거
2. 자막 수집 완료 후 AI 요약 Job 실행
3. OpenAI gpt-4o-mini 호출
4. DB 저장 (ai_summary jsonb)
5. 클라이언트 폴링 or Realtime으로 감지
6. 스켈레톤 → 콘텐츠 전환

## AI 프롬프트 방향
- 언어: 영상 언어 감지 후 한국어 출력
- 톤: 간결, 개인 학습 노트 스타일
- 프롬프트 원본: docs/references/ai-prompt.md

## 엣지 케이스
- 자막 없음: "자막이 없어 요약을 생성할 수 없습니다"
- 자막 < 500자: 요약 스킵, 원문 표시
- API 실패: 재시도 버튼
- 비유튜브: 본문 크롤링 후 요약 시도
