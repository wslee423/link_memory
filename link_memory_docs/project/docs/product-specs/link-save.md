# Feature: 링크 저장 + 태그

## 기능 설명
URL을 입력하면 저장되고, 이후 언제든 태그를 붙이거나 수정할 수 있다.

## 사용자 시나리오
1. 상단 입력창에 URL 붙여넣기
2. 저장 버튼 클릭
3. 자동으로 메타데이터 수집 시작
4. 카드 형태로 리스트에 추가 (스켈레톤 → 콘텐츠)
5. 태그 입력 가능 (저장 직후 or 나중에)
6. 태그 수정/삭제 언제든 가능

## UI 동작
- 상단 고정 입력창: URL 입력 + 저장 버튼
- 저장 즉시 카드 추가 (스켈레톤 로딩)
- 카드 클릭 시 상세 패널 열림
- 태그 자동완성: GET /api/tags 전체 목록 → 클라이언트 필터링
- 쉼표 or 엔터로 태그 구분

## 엣지 케이스
- 중복 URL: "이미 저장된 링크입니다" + 기존 카드 포커스
- 유효하지 않은 URL: 저장 버튼 비활성화
- 비유튜브 URL: OGP로 제목/썸네일 수집, 채널명/날짜 없음
- 메타데이터 실패: 제목=URL, 썸네일=기본 이미지
