# Feature: 유튜브 메타데이터 자동 수집

## 기능 설명
유튜브 URL 저장 시 자동으로 메타데이터를 수집해 카드에 표시한다.
자막은 AI 요약용으로 백엔드에만 저장 (클라이언트 미노출).

## 수집 항목
| 항목 | 카드 표시 | 용도 |
|------|-----------|------|
| 제목 | ✅ | 카드 타이틀 |
| 썸네일 | ✅ | 카드 이미지 |
| 채널명 | ✅ | 서브텍스트 |
| 업로드 날짜 | ✅ | 서브텍스트 |
| 자막 텍스트 | ❌ (비표시) | AI 요약 소스 — DB만 저장 |

## 수집 방식
- YouTube oEmbed API: 제목, 썸네일, 채널명
- YouTube Data API v3: 업로드 날짜
- YouTube Transcript API: 자막

## 엣지 케이스
- 자막 없는 영상: AI 요약 스킵, "자막 없음" 표시
- 비공개/삭제 영상: "불러올 수 없는 영상" 처리
- 비유튜브 URL: OGP 메타태그 폴백

- [ ] 자막 DB 저장 (클라이언트 미노출)
- [ ] 비유튜브 OGP 폴백
- [ ] 비공개/삭제 영상 에러 처리
