# Feature: 메모 작성

## 기능 설명
카드 상세 패널 안에서 plain text 메모를 자유롭게 작성/수정할 수 있다.

## UI 동작
```
[ 썸네일 + 제목 + 채널명 + 날짜 ]
[ AI 요약 (좌)        | 메모 (우)    ]
[ 태그                              ]
```
- 플레이스홀더: "이 영상에 대한 메모를 남겨보세요"
- 클릭 즉시 편집 모드 (별도 버튼 없음)
- 자동 저장: 1.5초 디바운스
- 상태 표시: "저장됨 ✓" or "저장 실패 - 재시도"
