# PRODUCT_SENSE.md

## Vision
내가 발견한 유튜브/웹 링크를 주제별로 아카이빙하고,
AI 요약·인사이트·메모를 붙여 나만의 지식 베이스로 만드는 개인 중심 웹 서비스.

## Target Users
- **Primary**: 서비스 소유자 본인 (관리자, 전체 권한)
- **Secondary**: 초대받은 소수 지인 — MVP 이후 지원 (읽기 전용)
  > 지인 공유는 MVP 스코프 외. Post-MVP P1 참고.

## Core User Journey
1. 유튜브 또는 웹 URL 발견
2. 사이트 상단 입력창에 URL 붙여넣기
3. AI가 자동으로 제목/썸네일 가져오고 요약 + 인사이트 생성
4. 사용자가 태그 분류 + 메모 추가
5. 나중에 주제별 탐색

## MVP Scope (v1)
- URL 저장 + 태그
- 유튜브 썸네일/제목/채널명/날짜 자동 수집
- AI 자동 요약 (OpenAI gpt-4o-mini)
- 메모 작성

## Out of Scope (Post-MVP)
- 지인 공유 → P1
- 크롬 익스텐션 → P2
- 고급 검색 → P1
- 멀티유저 → P3
