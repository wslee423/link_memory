# Design Core Beliefs

## 철학
- **정보 밀도 우선**: 목록과 상세를 한 화면에서 동시에
- **노이즈 제거**: 필요한 정보만 표면에, 나머지는 상세 패널로
- **빠른 입력**: URL 붙여넣기 → 저장 클릭 1번
- **다크 퍼스트**: 모든 컴포넌트 다크모드 기준

## 벤치마킹
- Learning Queue (learning-queue-bay.vercel.app)
  - 2패널 레이아웃, 상단 태그 가로 스크롤, 카드 클릭 → 우측 상세
