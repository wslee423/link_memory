# Component Spec

## 1. Header (sticky)
- 좌: 서비스 로고
- 중앙: URL 입력창 (placeholder: "유튜브 또는 웹 URL을 붙여넣으세요")
- 우: 저장 버튼

## 2. Tag Bar
- "전체" 태그 항상 첫 번째 고정
- 사용자 태그 가나다순
- 선택: indigo-500 배경
- 가로 스크롤, 스크롤바 숨김

## 3. Link Card
- 썸네일 (좌, 고정 크기)
- 제목 (최대 2줄 말줄임)
- 채널명 · 날짜 (서브텍스트)
- 태그 뱃지 (최대 3개, 초과 시 +N)
- 선택 상태: 좌측 indigo 보더 + 배경 변경
- AI 생성 중: shimmer 스켈레톤

## 4. Detail Panel
- 썸네일 (클릭 시 유튜브 새 탭)
- 제목 + 채널명 + 날짜
- AI 요약 (좌):
  - 생성 중: shimmer 스켈레톤
  - 완료: 3줄 요약 + 인사이트 불릿 (fade-in)
  - 실패: "요약 생성 실패 - 재시도" 버튼
- 메모 (우):
  - plain text textarea
  - 클릭 즉시 편집
  - 1.5초 자동저장 + "저장됨 ✓"
- 태그 섹션: 뱃지 + 삭제 + 추가 입력

## 5. Empty State
- 링크 없음: "아직 저장된 링크가 없어요. URL을 붙여넣어 시작해보세요 🔖"
- 태그 필터 결과 없음: "'[태그]' 태그가 달린 링크가 없어요"
