# Layout Spec

## 페이지 목록
| 경로 | 설명 |
|------|------|
| / | 메인 (링크 목록 + 상세 패널) |
| /login | 로그인 |

## 메인 페이지 구조
```
┌──────────────────────────────────────────────┐
│  HEADER: 로고 | URL 입력창 | 저장 버튼 (sticky)│
├──────────────────────────────────────────────┤
│  TAG BAR: [전체][claude code][notion]→  가로스크롤│
├─────────────────────┬────────────────────────┤
│  LEFT (40%)         │  RIGHT (60%)           │
│  링크 카드 목록      │  선택된 카드 상세        │
│  (최신순 고정)       │                        │
│  ┌───────────────┐  │  [ 썸네일 ]            │
│  │썸네일 제목     │  │  제목                  │
│  │       채널·날짜│  │  채널명 · 날짜         │
│  │       태그    │  │  ────────────────      │
│  └───────────────┘  │  AI 요약   │  메모     │
│  ┌───────────────┐  │  3줄 요약  │  (입력)   │
│  │ ...           │  │  인사이트• │           │
│  └───────────────┘  │  ────────────────      │
│                     │  태그: [t1][t2] +      │
└─────────────────────┴────────────────────────┘
```

## 반응형
| 브레이크포인트 | 레이아웃 |
|--------------|---------|
| > 1024px | 2패널 (40/60) |
| 768~1024px | 2패널 (50/50) |
| < 768px | 목록만 → 카드 탭 시 상세 풀스크린 |
